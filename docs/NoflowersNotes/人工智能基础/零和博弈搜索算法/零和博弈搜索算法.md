### alpha-beta 剪枝

$\alpha$: MAX 在该路径到跟上的当前最大取值 -- 下界;  
$\beta$: MIN 在该路径到根上的当前最小取值 -- 上界。

    for each successor of state:
        v = max(v, min-value(successor, alpha, beta)
        if v >= beta return v
        alpha = max(alpha, v)
    return v
    
def min-value(state, alpha, beta):
    v = - infty
    for each successor of state:
        v = min(v, max-value(successor, alpha, beta)
        if v <= alpha return v
        beta = min(beta, v)
    return v
