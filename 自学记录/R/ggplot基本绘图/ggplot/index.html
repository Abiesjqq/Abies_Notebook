    ggplot2 基础结构

    ggplot 的语法基于 图形语法 (Grammar of Graphics)：

    S
    1
    ggplot(data, aes(x, y)) + geom_xxx() + ...

    • data：数据集

    • aes()：映射（x 轴、y 轴、颜色、大小、形状等

    • geom_xxx()：几何对象（点、线、柱状图……）

    • +：叠加图层

# 准备数据：第一行 = 最大值，第二行 = 最小值
df <- data.frame(
  A = c(10, 0, 8, 3),
  B = c(10, 0, 7, 4),
  C = c(10, 0, 6, 2),
  D = c(10, 0, 9, 5),
  E = c(10, 0, 7, 3)
)

# 画雷达图
radarchart(df,
           axistype = 1,
           pcol = c("red", "blue"),
           plwd = 2,
           plty = 1,
           cglcol = "grey", cglty = 1, axislabcol = "grey",
           vlcex = 0.8)
