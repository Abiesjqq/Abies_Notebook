    List 的复杂度分析

    Python 的 list 是用 一块连续的内存空间 存储元素的：

    • 内部实际是一个 指针数组（即 PyObject** 指针数组），每个元素指向一个 Python 对象。
    • 所以列表本身存的是对象的引用，不是对象本身。
    • 所以 Python 的 list 可以包含不同类型的数据（因为存的是指针）。
